## What We Are Building

**Influencer Michelin** is a semi-automated Chrome extension that turns brand keywords into scored, verified influencer shortlists.

### Core Capabilities

1. **Account Collection** — Keyword, hashtag, and similar-account search to discover and save candidates directly from Instagram.
2. **Profile Extraction** — Automated capture of follower count, bio, recent posts, and engagement metrics.
3. **Scoring Engine** — Authenticity score (fake-follower detection), brand-fit score (content relevance), and ad-saturation flag.
4. **Review Workflow** — Notes, tags, exclude/archive actions for team-based candidate curation.
5. **Candidate Database** — Searchable, filterable, sortable list with automatic deduplication.

### Why This Matters

Brands waste hours manually vetting influencers across tabs and spreadsheets. Influencer Michelin collapses that workflow into a single extension — collect, score, review, export — so marketing teams spend time on partnerships, not research.

## Target Users

- Brand marketers managing influencer campaigns
- Agency teams scouting creators for multiple clients
- Solo DTC founders seeking their first creator partnerships

## Tech Stack

- Chrome Extension (Manifest V3)
- Instagram data extraction via browser-native methods
- Local-first candidate storage with optional cloud sync
